# Dan's Calculator

A simple, elegant web-based calculator for basic arithmetic operations.

## Features

- ✨ Clean and modern UI
- ➕ Addition, subtraction, multiplication, and division
- ⌨️ Keyboard support
- 🎯 Simple and intuitive interface
- 📱 Responsive design

## How to Use

1. Open `index.html` in your web browser
2. Click on numbers to input values
3. Click on operators (+, -, *, /) to perform operations
4. Click the = button to calculate the result
5. Use C to clear the display

## Keyboard Shortcuts

- **Numbers**: 0-9
- **Operators**: +, -, *, /
- **Decimal**: .
- **Calculate**: Enter or =
- **Delete**: Backspace
- **Clear**: Escape

## Files

- `index.html` - Calculator structure
- `style.css` - Styling and layout
- `script.js` - Calculator logic and functionality
